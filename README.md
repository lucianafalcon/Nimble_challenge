# Nimble Gravity — Fullstack Challenge

Mini aplicación en React que consume la API de Nimble Gravity para postularse a posiciones abiertas.

---

## Descripción

Esta aplicación permite:  

- Obtener la lista de posiciones abiertas desde la API.
- Mostrar cada posición con un input para ingresar la URL del repositorio de GitHub.
- Enviar la postulación a la posición seleccionada mediante la API.
- Manejar estados de carga y errores de manera sencilla.
- Presentar un diseño limpio y funcional, sin necesidad de librerías de UI externas.

---

## Tecnologías

- React 18+
- JavaScript / JSX
- Fetch API para llamadas HTTP
- CSS simple para estilos
